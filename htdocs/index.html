Examples

Below is an example of setting up my page title and introduction to be editable, using "name" and "type" attribute of text:
<h2><lk-var name="demos.demo-7324.title" type="text"></lk-var></h2>
<p><lk-var name="demos.demo-7324.intro" type="text"></lk-var></p>

Using booleans is simular to text except we are changing the "type" to boolean, giving us checkbox for selecting true or false:
<lk-var name="demos.demo-7324.publish" type="boolean"></lk-var>

Publish Article:
To use a select menu, we change the "type" to select and add the attribute "options", there are a few different ways to format options, i have used space seperated values, using camelCase to give spaces
<lk-var name="demos.demo-7324.layout" type="select" options="OneColumn TwoColumn ThreeColumn"></lk-var>

Page Layout:
The standard list, using the "type" of list, outputs a lsit of an array with a delete option for each item.
<lk-var name="demos.demo-7324.list" type="list"></lk-var>
To expand on the standard list, we can create custom lists, for example a todo list
Inside of our list from above we can add a custom template such as :
<lk-var model="item.title" type="text"></lk-var> <span class="delete-item"></span>
Doing so, we can access the text of title and make edits. We also needed to add the delete icon manually as its now a custom template
To add an item to the todolist, a button was added and a custom function written in the controller [x]
To expand further, instead of deleting items off our list, we want to set themt to completed.
Adding a boolean value of complete to the item schema and the following code to our template:
<lk-var model="item.complete" type="boolean"></lk-var>
we can now use a checkbox to set an item on the todo list to complete or incomplete
